Level 2 – Web Development & Designing Projects

A collection of four Level-2 web development tasks built using HTML, CSS, and JavaScript.
Each project strengthens core frontend skills—UI design, interactivity, DOM manipulation, responsiveness, and logical problem-solving.

🚀 1. Calculator

A fully functional calculator with a clean UI and grid-based button layout.
It performs basic mathematical operations such as addition, subtraction, multiplication, and division.
Built using event listeners, conditional logic, and operator handling, ensuring smooth user interaction and accurate results.

📝 2. Tribute Page

A simple, elegant webpage created using HTML & CSS to honor someone you admire.
Includes well-structured content, meaningful text, images, and visually appealing styling.
Typography, layout, and colors are used effectively to deliver a clean and expressive design.

📌 3. Basic To-Do Web App

A functional to-do application allowing users to add, edit, delete, and manage tasks.
Features include:

Add new tasks

Mark tasks as Completed

Manage Pending and Completed lists

(Optional) Add timestamps for task creation and completion
A great project for practicing DOM manipulation and user-state management.

🔐 4. Login Authentication System

A simple authentication system enabling users to register, login, and access a secured page.
Built using JavaScript logic to validate credentials and manage user sessions.
Demonstrates understanding of form handling and basic authentication workflows.

🛠️ Tech Stack

HTML – Structure

CSS – Styling & Layout

JavaScript – Interactivity & Logic

📂 Purpose

This repository showcases Level-2 frontend development skills through practical mini-projects that combine design, logic, usability, and clean UI patterns.
